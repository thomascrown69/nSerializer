nSerializer
===========

What is this?
Just a small lightweight serialising library mostly for networking, crossplatform ( Win, Linux, Mac, Android, iOS). 


Tested and compiled with:
-Visual Studio 2012 in Windows 7
-XCode - in Mac OS X Maverick
